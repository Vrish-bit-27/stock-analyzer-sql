# Stock Analyzer — Python + SQL (Google Colab)

## Overview
This project downloads real stock price data using **yfinance**, stores it inside a **SQLite database (SQL)**, and runs SQL queries to analyze the stock.

It was fully built in **Google Colab**.

---

## Tech Used
- Python
- Pandas
- SQLite (SQL)
- yfinance
- Google Colab

---

## Why this project matters
This is similar to how banks and trading teams:
- store market data
- run SQL queries
- generate insights before investment decisions

---

## What the project does
✔ Downloads daily stock price data  
✔ Stores it in a SQL table  
✔ Runs SQL queries to find:
- recent closing prices
- average closing price
- highest price
- lowest price  

✔ (Optional) Plots the stock price trend

---

## How to run (Colab)
1. Open Google Colab  
2. Paste the notebook code  
3. Install yfinance  
4. Run all cells  

---

## Example SQL insights
- Last 5 closing prices  
- Average closing price  
- Highest price date  
- Lowest price date
